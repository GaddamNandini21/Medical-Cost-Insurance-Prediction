# Medical Cost Insurance Prediction

## Project Overview

Medical Cost Insurance Prediction is a Machine Learning project that predicts the medical insurance charges of an individual based on various factors such as age, BMI, smoking habits, number of children, gender, and region.

The objective of this project is to build a regression model that can estimate insurance costs accurately.

---

## Problem Statement

Medical insurance costs vary depending on several personal and health-related factors. Predicting these costs manually is difficult.

This project uses Machine Learning techniques to predict insurance charges based on historical data.

---

## Dataset

The dataset contains the following attributes:

- Age
- Gender
- BMI
- Number of Children
- Smoker
- Region
- Insurance Charges
